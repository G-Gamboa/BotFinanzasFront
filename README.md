# BotFinanzas - Frontend

Este repositorio contiene el desarrollo de la interfaz de usuario para BotFinanzas, una plataforma orientada a la gestión y visualización de finanzas personales. El proyecto se centra en ofrecer una experiencia de usuario fluida mediante una arquitectura de componentes reactivos y un diseño basado en la claridad de los datos.

## Propósito del Proyecto

El objetivo de esta aplicación es servir como el cliente principal para interactuar con un motor de análisis financiero. La interfaz permite centralizar la entrada de datos (gastos e ingresos) y transformar esa información en reportes visuales comprensibles, eliminando la complejidad técnica de las hojas de cálculo tradicionales.

## Características Funcionales

### Gestión Conversacional
El frontend integra un módulo de chat diseñado para procesar la interacción con el bot financiero, permitiendo la entrada de datos de forma natural y la consulta de estados de cuenta sin navegar por menús complejos.

### Visualización de Indicadores
Implementación de tableros con gráficas dinámicas que reflejan la distribución de gastos, el ahorro mensual y la evolución del patrimonio. Se prioriza el uso de componentes que permitan la actualización de datos en tiempo real.

### Arquitectura de Interfaz
Diseño modular que facilita la navegación entre el historial de transacciones, la configuración de presupuestos y el panel principal de resumen financiero.

## Stack Tecnológico

La selección de herramientas se ha realizado buscando un equilibrio entre rendimiento y mantenibilidad del código:

*   **React:** Biblioteca base para la construcción de la interfaz mediante componentes.
*   **TypeScript:** Implementado para añadir tipado estático, reduciendo errores en tiempo de desarrollo y mejorando la documentación interna del código.
*   **Vite:** Herramienta de construcción y servidor de desarrollo para optimizar los tiempos de recarga y el empaquetado final.
*   **Tailwind CSS:** Framework de diseño basado en utilidades para la construcción de una interfaz consistente y adaptable a diferentes resoluciones de pantalla.
*   **Lucide React:** Set de iconos vectoriales para la señalética del sistema.

## Enfoque de Desarrollo

El desarrollo se rige bajo los siguientes principios técnicos:

1.  **Componentización:** Creación de elementos UI reutilizables para mantener la coherencia visual y facilitar el escalado del proyecto.
2.  **Responsividad:** La interfaz está optimizada para ser funcional tanto en entornos de escritorio como en dispositivos móviles, adaptando los flujos de interacción según el dispositivo.
3.  **Manejo de Estado:** Gestión eficiente del flujo de datos entre la interacción del usuario con el bot y la actualización de las métricas en los tableros.

## Estado Actual

El proyecto se encuentra en una fase de desarrollo activo, centrada en la integración de servicios de backend y el refinamiento de la lógica de presentación de datos complejos.
